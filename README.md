# Everlit Candle

**Some lights should never go out.**

A digital prayer candle platform where you can light a unique candle that lives permanently on-chain. Return to your candle anytime you need a quiet moment of peace.

## About

Everlit Candle creates eternal digital candles connected to prayers, memories, hopes, and intentions. Each candle is minted as an NFT on the Solana blockchain, creating a permanent, ownable record of your personal reflection.

## Features

- 🔥 **Light a Candle** — Create a unique digital candle with your prayer or intention
- 🕯️ **Eternal Flame** — Your candle never expires, living permanently on-chain
- 🧘 **Moment of Peace** — Fullscreen mode for distraction-free reflection
- 🕸️ **Candle Wall** — Anonymous public gallery of all lit candles
- 💳 **Simple Payments** — Credit card or crypto, no technical knowledge needed
- 🔐 **True Ownership** — Your candle belongs to you, transferable to any wallet

## Tech Stack

- **Frontend**: Pure HTML/CSS/JS (GitHub Pages)
- **Backend**: Firebase (Auth + Firestore)
- **Payments**: Crossmint (fiat + crypto)
- **Blockchain**: Solana (Devnet → Mainnet)

## Development

This is a static website designed to run on GitHub Pages with Firebase backend integration.

### Local Development

```bash
# Serve locally
python -m http.server 8080
```

### Deployment

1. Push to GitHub
2. Enable GitHub Pages in repository settings
3. Configure custom domain (everlitcandle.com)
4. Setup Firebase project for backend

## AI-Agentic Design

This website is built with AI-agentic principles:

- **Semantic data attributes** for easy navigation (`data-section`, `data-action`)
- **Structured metadata** for machine readability
- **Predictable API** exposed via `window.EverlitAPI`
